# Cloud Storage Backend Project

## Description

This project is a simple backend system inspired by Google Drive. It includes basic APIs for user registration, login, and file handling.

## Features

* User Registration
* User Login
* File Create & List APIs

## Tech Stack

* Node.js
* Express.js

## How to Run

1. Install dependencies:
   npm install

2. Start server:
   npm start

3. Open browser:
   http://localhost:3000

## Note

This is a basic version for learning and submission purposes.
